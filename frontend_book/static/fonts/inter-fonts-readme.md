# Inter Font Files

This directory is intended to contain the Inter font files for local hosting.

## How to Obtain Inter Fonts

1. **Download from Google Fonts**: Visit [fonts.google.com/specimen/Inter](https://fonts.google.com/specimen/Inter) and download the font files
2. **Download from GitHub**: Clone the repository at [github.com/rsms/inter](https://github.com/rsms/inter)
3. **Install via NPM**: `npm install inter-ui` (if using as a package)

## Required Font Weights
- Inter-Regular.woff2 (400)
- Inter-Medium.woff2 (500)
- Inter-SemiBold.woff2 (600)
- Inter-Bold.woff2 (700)

## Usage
After obtaining the font files, place them in this directory and update your CSS to reference them locally.
