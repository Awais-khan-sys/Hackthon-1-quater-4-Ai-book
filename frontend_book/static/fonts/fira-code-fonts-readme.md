# Fira Code Font Files

This directory is intended to contain the Fira Code font files for local hosting.

## How to Obtain Fira Code Fonts

1. **Download from GitHub**: Visit [github.com/tonsky/FiraCode](https://github.com/tonsky/FiraCode) and download the font files from the releases
2. **Download from Google Fonts**: Visit [fonts.google.com/specimen/Fira+Code](https://fonts.google.com/specimen/Fira+Code) and download the font files
3. **Install via NPM**: `npm install fira-code` (if using as a package)

## Required Font Weights
- FiraCode-Regular.woff2 (400)
- FiraCode-Medium.woff2 (500)
- FiraCode-SemiBold.woff2 (600)
- FiraCode-Bold.woff2 (700)

## Usage
After obtaining the font files, place them in this directory and update your CSS to reference them locally.
